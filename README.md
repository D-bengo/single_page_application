#  Dictionary Web App

## Overview

The **Dictionary Web App** is a modern, single-page web application (SPA) that allows users to search for English words and get detailed information including:

- Definitions
- Synonyms
- Pronunciation text
- Audio pronunciation

It uses the **Free Dictionary API** to fetch live word data and is built with **HTML, CSS, and JavaScript**.

> This project is designed to demonstrate real-world API integration, DOM manipulation, and responsive web design.
